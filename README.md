# Global Money Intercepter

**App global money entercper is a one stop app** — a simple, browser-based currency converter covering 20 major world currencies.

## 🌐 Run it now

Open `index.html` directly in any web browser — no build step, no server required.

### Quick start with a local server

```bash
# Python 3
python -m http.server 8080
# then open http://localhost:8080
```

```bash
# Node.js (npx)
npx serve .
```

### GitHub Pages

Enable **GitHub Pages** in the repository settings (source: `main` branch, root `/`) to get a public URL:

```
https://<username>.github.io/Global-money-entercper-by-Gemini-/
```

## ✨ Features

- Convert between 20 global currencies (USD, EUR, GBP, JPY, CNY, INR, NGN, AED, and more)
- Swap currencies with one click
- Live display of popular exchange rates (USD base)
- Works entirely offline — no API key needed
